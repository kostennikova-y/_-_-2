# Calculator (Git Laboratory Work)

## Описание
Данный проект представляет собой консольное приложение «Калькулятор», разработанное на языке Python
с использованием системы контроля версий Git.

## Функциональность
Приложение поддерживает следующие операции:
- Сложение
- Вычитание
- Умножение
- Деление
- Остаток от деления
- Sin
- Cos
- Возведение в степень
- Квадратный корень
- Округление в меньшую сторону (floor)
- Округление в большую сторону (ceil)
- Работа с памятью (MC, M+, MR)

## Структура проекта

calculator/
├── calculator.py # Математические операции
├── memory.py # Работа с памятью
├── main.py # Тестирование приложения
├── README.md
└── .gitignore


## Используемые технологии
- Python 3
- Git
- PyCharm
- Windows

## Работа с Git
- Каждая функция реализована в отдельной ветке
- Использованы коммиты для фиксации изменений
- Выполнены слияния веток (merge)
- Продемонстрировано разрешение merge-конфликта
